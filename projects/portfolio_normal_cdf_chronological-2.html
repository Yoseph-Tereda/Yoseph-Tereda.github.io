Very Simply Explicitly Invertible Approximation for the Normal CDF

Project Overview

This project presents a new, very simply explicitly invertible approximation for the standard normal cumulative distribution function (CDF). The goal is to obtain a mathematical model that is analytically invertible, enabling efficient Gaussian-based transformations in simulation, data science, and engineering workflows.

The work includes formulation of the approximation, nonlinear least-squares parameter fitting, error analysis, and evaluation against existing approximations.

Proposed Approximation Function

The paper introduces a deliberately simple, explicitly invertible approximation based on five adjustable parameters.

\[ F(z;\mathbf{c}) = \Bigg( 1 + c_1 \Big[\, \ln\big(1 + e^{-z / c_5 + c_3}\big) \,\Big]^{c_2} \Bigg)^{-c_4} \]

Here \(\mathbf{c}=(c_1,c_2,c_3,c_4,c_5)\) are design parameters chosen by optimization. The inner \(\ln(1+e^{\cdot})\) term provides smooth saturation similar to a softplus/logistic transform, the power \(c_2\) and multiplier \(c_1\) allow flexible shaping, and the outer exponent \(-c_4\) maps the expression into \([0,1]\).

Least–Squares Parameter Estimation

Parameters are obtained by minimizing the ordinary least-squares residual between the approximation and the true standard normal CDF values at sampled points \(z_i\):

\[ \min_{\mathbf{c}} \; J(\mathbf{c}) \;=\; \sum_{i=1}^N \big[\, F(z_i;\mathbf{c}) - \Phi(z_i) \,\big]^2 \]

In practice the fit is performed for \(z\ge 0\) and extended to negative \(z\) via the symmetry \(\Phi(-z)=1-\Phi(z)\). Constrained fits enforcing \(F(0;\mathbf{c})=0.5\) were also considered to preserve the CDF midpoint exactly. Global optimization (multiple random starts) was used to avoid poor local minima.

Key Contributions

  • Formulated a compact, explicitly invertible five-parameter approximation.
  • Designed and executed a least-squares fitting procedure with global search and BARON to find robust parameter sets.
  • Performed sensitivity analysis to identify dominant parameters (c3 and c5).
  • Reported multiple fits (constrained and unconstrained) with MAE down to 2.73e–05.
  • Validated inverse accuracy and demonstrated competitive runtime in an application example.

Applications

This approximation is suitable for Monte Carlo simulation, random-variable transformations requiring an explicit inverse, correlation-preserving stochastic modeling, and other contexts where fast, invertible Gaussian CDF evaluations are needed.
