Numerical Solutions for Sub-Diffusion Problems

Yoseph Tiruwuha Tereda, M.Sc. — King Fahd University of Petroleum & Minerals

Project Overview

This project investigates the numerical approximation of time–fractional sub-diffusion equations, which arise in the modeling of anomalous transport phenomena in heterogeneous and memory-dependent media. These processes are characterized by non-local temporal behavior and weak solution regularity near the initial time, posing significant challenges for classical numerical time-stepping methods.

To overcome these difficulties, implicit time integration schemes based on the Backward Euler and Crank–Nicolson methods are combined with graded temporal meshes that refine the time step near t = 0. This strategy compensates for the singular behavior of fractional derivatives and restores optimal or near-optimal convergence rates.

Rigorous stability and error analyses are developed and validated through numerical experiments. Spatial discretization is carried out using finite element methods, yielding fully discrete schemes suitable for practical simulation of sub-diffusive processes.

Governing Equation

We consider the following time–fractional reaction–diffusion problem: 

∂u(x,t)/∂t + D_t^{α} ( -∇·(κ(x)∇u(x,t)) + c(x)u(x,t) ) = f(x,t),
        for x ∈ Ω, 0 < t ≤ T,

subject to the initial and boundary conditions 

u(x,0) = u₀(x),              for x ∈ Ω,
u(x,t) = 0,                 for x ∈ ∂Ω, 0 < t ≤ T.

Here:

  • Ω ⊂ ℝd (d = 1,2,3) is a bounded spatial domain,
  • Dtα denotes the Caputo fractional derivative of order 0 < α < 1,
  • κ(x) > 0 is the spatially varying diffusion coefficient,
  • c(x) ≥ 0 represents a reaction term,
  • f(x,t) is a given source function,
  • u₀(x) is the prescribed initial condition.

This formulation captures both anomalous diffusion effects and local reaction mechanisms, and exhibits weak solution regularity near t = 0.

Conclusions

  • Weak initial-time regularity significantly affects numerical accuracy.
  • Graded temporal meshes restore optimal convergence behavior.
  • Crank–Nicolson achieves higher accuracy than Backward Euler.
  • Finite element discretization yields robust fully discrete schemes.
  • Numerical experiments confirm theoretical convergence results.
