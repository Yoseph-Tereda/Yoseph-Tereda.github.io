Fuzzy Logic Controller for a Physical Distance Detector

Project Summary

This project presents a theoretical fuzzy logic‚Äìbased controller designed to estimate and regulate physical distance between individuals during pandemic scenarios such as COVID-19. The system transforms raw distance measurements into linguistic variables using membership functions and applies interpretable rule-based reasoning to generate alarm responses. By avoiding rigid thresholding, the controller provides smooth, explainable, and noise-tolerant decisions suitable for real-time proximity monitoring in wearable or embedded devices.

Mathematical Model

1. Fuzzification

Crisp distance input $d$ is mapped to: $$ \mu_{\text{near}}(d),\; \mu_{\text{medium}}(d),\; \mu_{\text{far}}(d) $$ $$ \mu_{\text{near}}(d) = \begin{cases} 1, & d \le a \\ \frac{b-d}{b-a}, & a < d < b \\ 0, & d \ge b \end{cases} $$
2. Rule Base
  • IF distance is very close ‚Üí alarm is high
  • IF distance is close ‚Üí alarm is medium
  • IF distance is far ‚Üí alarm is low
  • IF distance is very far ‚Üí alarm is very low
3. Aggregation $$ \mu_{\text{output}}(x) = \max_i (\alpha_i \mu_i(x)) $$ 4. Defuzzification $$ y = \frac{\int x \mu_{\text{output}}(x)\,dx}{\int \mu_{\text{output}}(x)\,dx} $$

Conclusion

This work demonstrates how fuzzy logic provides an effective and interpretable framework for physical distance detection. The proposed controller enables smooth alarm transitions, robustness to noisy measurements, and transparent decision-making. If implemented as a wearable or embedded system, it could support safer public interactions, enforce distancing policies, and reduce infection risk during health crises.
